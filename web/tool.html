Binaries

Download the spdiff tool for Linux: spdiff.opt.

Source code

The source code for spdiff is currently not available online. Eventually, it will be made available as well as instructions for compiling the source code.

System requirements

The binary provided above spdiff has been compiled to native x86 code using dynamic linking. It should run just fine on any modern Linux distribution.

Binaries for windows can be provided if the need arises. Contact me at email.

The memory usage of spdiff depends on the size of the dataset provided to it. For 100.000 lines of code in the dataset, the tool uses around 30-40 MB.

Installation

Simply put the binary somewhere in your search path.

You can read about how to run the tool in the manual: manual.
